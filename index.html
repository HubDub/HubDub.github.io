Say What? Professionally Speaking

SayWhat? Professionally Speaking is an app to help managers pick the right words when filling out performance evaluations. The manager can search for new phrases based on topic and whether the employee is poor, average, or competent, save their phrases or create new ones for future use. This app was created using Angular & bootstrap and Firebase to store the phrases.

  • GitHub: SayWhat?

Pinterested?

Pinterested? is a mock Pinterest app that allows the user to log in, create new pins, create new boards, save any pins to personal boards, and to view the boards with the pins. This app is built in angular with materialize using Firebase on the backend.

  • Github: Pinterested?

Benefits Portal

Benefits is a portal for employees and HR staff to log in to see their benefits on one page. HR staff can also see a dashboard of all employees benefits. This app is a ASP.NET MVC Core application created in Visual Studio in the language of C#.

  • Github: Benefits Portal
